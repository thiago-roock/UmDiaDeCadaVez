![CodeQL](https://github.com/thiago-roock/UmDiaDeCadaVez/workflows/CodeQL/badge.svg)

O que é o melhor que você pode fazer hoje?

Um lembrete simples, direto e diário: um dia de cada vez.

---

## ✨ Sobre o projeto

Uma aplicação web minimalista focada em consciência diária, constância e progresso.

A proposta é simples:

- Mostrar quantos dias já passaram no ano  
- Mostrar quantos ainda faltam  
- Exibir o progresso  
- Trazer uma frase do dia  
- Permitir compartilhar esse momento  

Tudo com uma experiência leve, rápida e sem distrações.

---

## 🚀 Funcionalidades

- Contagem dinâmica de dias (mais um / menos um)
- Barra de progresso do ano
- Frase do dia automática
- Modo claro / escuro
- Compartilhamento da tela como imagem
- Layout responsivo (mobile-first)

---

## 🧱 Arquitetura

Projeto estruturado com foco em simplicidade, clareza e manutenção.

CSS:
- estilos.css → base global (tokens, reset, componentes)
- mobile.css → layout base mobile-first
- site.css → melhorias para telas maiores

JavaScript:
- progresso.js → cálculo de dias e progresso
- frases.js → frase do dia
- data.js → controle de data
- gerarImagem.js → exportação da tela
- darkMode.js → tema claro/escuro
- site.js → inicialização geral
